# Introduction

Welcome to the Official Documentation for Burokku, you will find useful resources here to help you get started with Burokku.

## What is Burokku?

A **flexible** and **easy-to-use** Web app that aim to help the average Web3 user like me to get exposure to the Web3 world and consume information in a more **intuitive** way.
